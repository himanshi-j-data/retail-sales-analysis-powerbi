# Retail Sales Analysis Dashboard
# Project Overview

This project analyzes retail transaction data to identify revenue trends, profit distribution, and customer performance.

# Tools Used

Python (Data Cleaning & Transformation)

SQL (Business Query Analysis)

Power BI (Dashboard Visualization)

# Key Insights

West region generated highest revenue

Technology category delivered maximum profit

Revenue peaked in Q4

Top customers contributed disproportionately to sales
